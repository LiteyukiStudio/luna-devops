# 账单与费用分析

账单页用于查看项目空间余额、消费概览、费用分析和账单流水。

页面顶部的项目空间选择是全页范围控制。切换范围后，余额卡片、本月分类消耗、费用分析、账单流水和用量记录都会按同一批项目空间重新统计。未选择项目空间时，普通用户默认查看与自己相关的项目空间；平台管理员默认查看全部项目空间。

平台管理员还可以在站点设置里配置现实货币单位和换算比例。账单页顶部概览会在 credits 后显示折算金额，例如 `1,012.24 Credits (1.01 CNY)`；这只是展示换算，底层结算仍使用 credits。

## 费用分析

费用分析按“项目空间 / 应用”聚合已结算用量，展示总费用以及构建、运行、存储、访问和其他分类费用。

如果一条历史用量没有关联到应用，页面会归到“未关联应用”。账单流水仍保留每一笔余额变动，适合审计；费用分析更适合判断哪个项目或应用正在产生主要开销。
